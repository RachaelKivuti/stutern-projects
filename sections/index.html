I have learnt that HTML stands for HyperText Markup Language. It is a standard markup language for web page creation. HTML is the building block of a web page, that is, It allows the creation and structure of sections, paragraphs, and links using HTML elements such as tags and attributes. It is used in web development.
Css stands for Cascading Style Sheets. It gives a website its look and layout. Along with HTML, CSS is fundamental to web design. Without it, websites would still be plain text on white backgrounds. It helps us to control the text color, font style, the spacing between paragraphs, sizing of columns, layout designs, and many more.

Padding is the inner space between the content and the border of your box.
Border is the perimeter of the box. Borders can be invisible or they could be a thick colored line like the green one pictured above.
Margin is the outer space (or lack of space) surrounding the box.
